# Emacsで絵文字を表示させるには？

[Symbola フォント](https://ja.fonts2u.com/symbola.%E3%83%95%E3%82%A9%E3%83%B3%E3%83%88)をインストールすれば自動で表示される（Emacs 27.1で確認）。ただし、表示はカラーではなく白黒。


# Emacsで絵文字を入力するには？

Mozcで変換させれば入力できる。

| 絵文字 | よみがな |
| --- | ---- |
| 😁  | えがお |
| 🌄  | たいよう |
| 🌔  | つき |

なんて楽ちんな世の中だ。
